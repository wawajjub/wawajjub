# 20243059 손주한   
1. top   
top명령어는 cpu 사용률과 메모리 사용량등의 os의 전반적인 상황을 보여준다.
```bash
$ top
```

2. ps   
ps명령어는 현재 실행중인 프로세스와 상태를 출력한다.
```bash
$ ps -ax
```
3. jobs
jobs명령어는 리눅스에서 백그라운드로 실행되고 있는 프로세스를 조회한다.
```bash
$ jobs
```
4.kill
kill명령어는 프로세스를 죽일 때 사용한다.
```bash
$ kill [option] <pid> [...]
```
